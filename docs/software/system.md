# System & Monitoring

Tools for keeping an eye on hardware health, performance, and system resources.

### MSI Center
| | |
|---|---|
| **WinGet ID** | Manual install |
| **Download** | [MSI Cubi 5 12M Support](https://www.msi.com/Business-Productivity-PC/Cubi-5-12M/support) |
| **Guide** | [MSI Center Info](https://www.msi.com/Landing/MSI-Center) |
| **Use Case** | MSI's system management tool. Fan control, power profiles, hardware monitoring. Downloaded from MSI support page as part of initial driver setup |

### CrystalDiskInfo
| | |
|---|---|
| **WinGet ID** | `CrystalDewWorld.CrystalDiskInfo` |
| **Download** | [crystalmark.info](https://crystalmark.info/en/software/crystaldiskinfo/) |
| **Guide** | [CrystalDiskInfo Docs](https://crystalmark.info/en/software/crystaldiskinfo/) |
| **Use Case** | Drive health monitor. Tracks S.M.A.R.T. data on all drives including externals. Will alert if a drive is at risk of failure |

### WizTree
| | |
|---|---|
| **WinGet ID** | `AntibodySoftware.WizTree` |
| **Download** | [diskanalyzer.com](https://www.diskanalyzer.com/) |
| **Guide** | [WizTree Help](https://www.diskanalyzer.com/help) |
| **Use Case** | Visual disk space analyser. Shows exactly where storage is being used across all drives. Essential for managing large media libraries |

### HWiNFO64
| | |
|---|---|
| **WinGet ID** | `REALiX.HWiNFO` |
| **Download** | [hwinfo.com](https://www.hwinfo.com/) |
| **Guide** | [HWiNFO Docs](https://www.hwinfo.com/help/) |
| **Use Case** | Deep hardware monitoring. CPU temps, fan speeds, voltages. More detailed than MSI Center |

### ProcessLasso
| | |
|---|---|
| **WinGet ID** | Manual install (licence held) |
| **Download** | [bitsum.com](https://bitsum.com/) |
| **Guide** | [ProcessLasso Docs](https://bitsum.com/processlasso-docs/) |
| **Use Case** | Process priority manager. ProBalance prevents any single process (e.g. Plex transcoding, downloads) from hogging CPU on the Cubi 5's limited cores |
